## How to run the development server

To run sass:
```shell
    $ sass --watch front/src/scss/components/index.scss:front/src/main.css 
```

To run elm development server:
```shell
    $ cd front
    $ elm-app start
```

To run haskell api:
```shell
    $ cd api
    $ runghc Main.hs
```
